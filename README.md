# Golfrowdy 🛺💥

Ein 3D-Browserspiel: Verwüste mit dem Golfcart einen 18-Loch-Golfplatz,
ohne dich vom Marshall erwischen zu lassen. Läuft am Desktop (Tastatur)
und auf dem iPad/Smartphone (Touch-Steuerung).

## So veröffentlichst du das Spiel auf GitHub Pages

1. Auf [github.com](https://github.com) einloggen und oben rechts **+ → New repository** wählen.
2. Namen vergeben (z. B. `golfrowdy`), **Public** auswählen, **Create repository**.
3. Auf der Repo-Seite **uploading an existing file** anklicken und **alle Dateien
   aus diesem Ordner** hineinziehen (`index.html`, den Ordner `assets`,
   `.nojekyll`, diese README) → **Commit changes**.
4. Im Repo: **Settings → Pages** → unter *Build and deployment*:
   Source **Deploy from a branch**, Branch **main** und Ordner **/ (root)** → **Save**.
5. Nach 1–2 Minuten ist das Spiel erreichbar unter:
   `https://DEIN-BENUTZERNAME.github.io/golfrowdy/`

Diese Adresse einfach auf dem iPad in Safari öffnen.
Tipp: **Teilen → Zum Home-Bildschirm** installiert das Spiel als Vollbild-App.

## Steuerung

| Desktop | iPad |
|---|---|
| W / ↑ – Gas | GAS-Button (rechts) |
| S / ↓ – Bremse/Rückwärts | BREMSE-Button |
| A / D bzw. ← / → – Lenken | Pfeil-Buttons (links) |
| Leertaste – Driften | DRIFT-Button |
